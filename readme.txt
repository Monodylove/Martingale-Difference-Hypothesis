本程序依赖三个头文件，需自己下载至计算机中并牢记存放地址：armadillo，stats-master，gcem-master
This program relies on three header files, which need to be downloaded to the computer and keep in mind the storage address: armadillo, stats-master, gcem-master 

以visual studio为例，
Take visual studio as an example, 
①：导航栏：生成->配置管理器，选择x64
②：导航栏：项目->属性，打开项目属性页
       “VC++目录->包含目录”添加“.\armadillo”文件夹下的“include”文件夹
       “VC++目录->包含目录”添加“.\stats-master”文件夹下的“include”文件夹
       “VC++目录->包含目录”添加“.\gcem-master”文件夹下的“include”文件夹
